# Rune Landing Page

Static Vite landing page for Rune, ready to deploy on Vercel.

## Local Development

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Production Build

```bash
npm run build
npm run preview
```

## Deploy On Vercel

1. Import the Rune GitHub repository in Vercel.
2. Set the project root directory to `site`.
3. Keep the build command as `npm run build`.
4. Keep the output directory as `dist`.

The same settings are also defined in `vercel.json`.
